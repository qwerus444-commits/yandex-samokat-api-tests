# Тестирование API Яндекс Самокат

## Описание
Проект содержит автотест для проверки получения заказа по треку.

## Структура проекта
- `configuration.py` – базовый URL и тестовые данные
- `api_methods.py` – функции для работы с API
- `test_order.py` – автотест
- `.gitignore` – список игнорируемых файлов
- `README.md` – описание проекта
